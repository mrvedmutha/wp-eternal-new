## 1. Mission Statement

Scaffold a reusable, fully typographic Gutenberg block (`wp-rig/editorial-text-section`) that renders an eyebrow label, a two-paragraph body, and a pull quote in an editorially-styled sticky-scroll section. The sticky behaviour is modelled on `homepage-brand-statement`; all design tokens (typography, color, spacing) come 1:1 from Figma node 694:4211.

## 5. Sticky Scroll Pattern (from homepage-brand-statement)

- Section wrapper: `position: relative; height: 800px; background: #f5f5f5`
- Spacer div (80px height, `aria-hidden`) sits above the sticky block
- Inner content: `position: sticky; top: 80px`
- GSAP + ScrollTrigger: word-reveal plays once on scroll enter, staggered sequentially — eyebrow words → body words → pull quote words
- **Mobile (≤ 1024px):** height auto, sticky removed, plain flow
- **Small mobile (≤ 450px):** body font-size drops to 24px/31px, tracking: 0

---

## 6. Editor UI (edit.js)

InspectorControls panel "Editorial Text Section" with four `TextareaControl` fields:

1. Eyebrow Label
2. Body — Paragraph 1
3. Body — Paragraph 2
4. Pull Quote

Canvas preview renders a static representation of all four zones with correct class names.
